# Technical test

## Description

This is a technical test for a job interview.
The app is available at [https://test-tech-stm.vercel.app/](https://test-tech-stm.vercel.app/)

## Installation

```bash
$ git clone git@github.com:FaureAlexis/test-tech-stm.git
$ cd test-tech-stm
$ npm install
```

## Running the app

### development
```bash
$ npm run dev
```

### production mode
```bash
$ npm run build
$ npm run start
```
